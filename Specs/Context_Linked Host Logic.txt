Prompt: Detecting and Working with Linked Model Hosts in Revit
When working with Revit family instances that can be hosted to elements in linked models, you need special handling to access the host geometry. The standard approach for local hosts will fail silently for linked hosts.
Detecting a Linked Model Host
Check if the fixture's host is a RevitLinkInstance:
csharpElement host = fixture.Host;

if (host is RevitLinkInstance linkInstance)
{
    // Host is in a linked model
}
else
{
    // Host is in the current document
}
Accessing Geometry from a Linked Host
For local hosts, you can directly get geometry:
csharpGeometryObject geomObj = host.GetGeometryObjectFromReference(hostFaceRef);
For linked hosts, you must:

Get the linked document from the RevitLinkInstance
Get the element in the linked document using LinkedElementId
Create a reference valid in the link's context using CreateReferenceInLink()
Get the geometry from that element

csharpif (host is RevitLinkInstance linkInstance)
{
    Document linkedDoc = linkInstance.GetLinkDocument();
    if (linkedDoc != null)
    {
        Element linkedElement = linkedDoc.GetElement(hostFaceRef.LinkedElementId);
        Reference linkRef = hostFaceRef.CreateReferenceInLink();
        GeometryObject geomObj = linkedElement?.GetGeometryObjectFromReference(linkRef);
    }
}
Transforming Coordinates from Link to Host
Geometry retrieved from a linked model is in the link's coordinate system. To use it in the host document, transform it using the link's total transform:
csharpTransform linkTransform = linkInstance.GetTotalTransform();

// For vectors (directions, normals):
XYZ transformedNormal = linkTransform.OfVector(originalNormal);

// For points:
XYZ transformedPoint = linkTransform.OfPoint(originalPoint);
Complete Example: Getting Host Face Normal
csharpprivate XYZ GetHostFaceNormal(FamilyInstance fixture)
{
    Reference hostFaceRef = fixture.HostFace;
    if (hostFaceRef == null)
        return null;

    Element host = fixture.Host;
    
    if (host is RevitLinkInstance linkInstance)
    {
        // Linked model host
        Document linkedDoc = linkInstance.GetLinkDocument();
        if (linkedDoc == null)
            return null;

        GeometryObject geomObj = linkedDoc.GetElement(hostFaceRef.LinkedElementId)
            ?.GetGeometryObjectFromReference(hostFaceRef.CreateReferenceInLink());

        if (geomObj is PlanarFace planarFace)
        {
            Transform linkTransform = linkInstance.GetTotalTransform();
            return linkTransform.OfVector(planarFace.FaceNormal);
        }
    }
    else
    {
        // Local host
        GeometryObject geomObj = host.GetGeometryObjectFromReference(hostFaceRef);
        
        if (geomObj is PlanarFace planarFace)
        {
            return planarFace.FaceNormal;
        }
    }

    return null;
}
Key Notes

GetLinkDocument() returns null if the link is unloaded
LinkedElementId gives the element ID within the linked document's context
CreateReferenceInLink() converts the reference to be valid in the linked document
Always transform vectors and points from link coordinates to host coordinates
Use OfVector() for directions/normals (ignores translation)
Use OfPoint() for locations (includes translation)
